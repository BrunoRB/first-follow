first-follow

Simple first and follow sets generator for context-free grammars.

Accepted input

  • Nonterminals should be separated from their derivations by -> or →, ex:
    • A -> B
    • B → y
  • Multiple symbols should be separated by blank spaces, ex:
    • X → yOne yTwo y3
    • yTwo → W thisIsConsideredAsingleSymbol
    • W → those are considered five symbols
  • Each set of rules must be in a newline and must not break into a new line or optionally multiple sets of rule can be placed at same line as long as all of them, with the exception of the last one, are terminated by a semicolon (;), ex:
    • A → y w B
    • B → C y; C → w
  • Multiple derivations can be stated in the same line using the "|" separator:
    • A → w y | k | f
  • For Epsilon derivations use ε or leave the derivation empty:
    • A → B | ε
    • B → | K
    • K →
